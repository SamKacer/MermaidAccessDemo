Keyboard shortcuts Reference

Note: if not working, make sure to be in focus mode for your screen reader.

Table navigation

  • Left/right arrow: move to previous/next column

  • Up/down arrow: when in a column with multiple items, move to previous/next item

  • Enter: navigates to associated table of selected item

(For example, when looking at an edge, will navigate to the node the edge connects to.)

  • Escape: collapses and exits out of the accessible viewer

  • Ctrl+Left/Right: Go backward/forward in history of visited tables

  • Shift+H: set current table as home

  • H: navigate to home table

  • F: bring up search dialogue

When in search dialogue

  • Type label of a node or edge you are looking for

  • Backspace: deletes last character of search query like when typing normally, but note other text editing shortcuts don’t work

  • Up/down arrow: move to previous/next search result

  • Enter: navigate to item’s associated table (for nodes it navigates to the node’s table, for edges it navigates to node table of its starting node)

  • Escape: exit search dialogue and return to normal view
